cmd folder -> here basically where is the executable file, and you can use it to run your project or to build your project and run it in another machine or server, and you can use it to run your project in production, where you can run it by go run cmd/main.go or go run cmd/anotherfile.go,inside the cmd folder where the name of our application per example create a folder called create-user and inside it create a file called main.go, and inside it write the code of your application, and you can run it by go run cmd/create-user/main.go, and you can build it by go build cmd/create-user/main.go, and you can run it by ./main, and you can run it in production by go install cmd/create-user/main.go, and you can run it by create-user
